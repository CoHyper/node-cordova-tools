# node-cordova-tools
> BETA


## Require
* NodeJS

## Require Optional
````shell
// js-eslint
npm install --save-dev eslint
 
// node-minify
// https://www.npmjs.com/package/node-minify
npm install --save-dev node-minify

// sass
gem install sass
 
````

## Create new project
copy in same of 'node_modules/'
```shell
cp -r node-cordova-tools/example/* ../../
```

## Structur
Settings in 'nodeCordovaTools.js' must in parent folder.

```
root/
|- node-cordova-tools/*
|- node_modules/*
|- .gitignore
|- .gitmodules
|- nodeCordovaTools.js
|- package.json
|- README.md
```
